Place your app icons here:

- icon-192.png      (192x192 PNG, used for Android home screen)
- icon-512.png      (512x512 PNG, used for larger displays / install prompts)
- maskable-192.png  (192x192 PNG with safe margins, used for maskable icons)
- maskable-512.png  (512x512 PNG with safe margins, used for maskable icons)

Until you add custom icons, the app will fall back to the default favicon for some contexts.
